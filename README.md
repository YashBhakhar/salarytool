# Salary Management Tool

## Tech Stack

- React
- TypeScript
- Tailwind
- Node.js
- Express
- SQLite
- React Query
- Vitest

## Features

- Employee CRUD
- Salary analytics
- Search/filtering
- Pagination
- HR dashboard

## Setup

npm install
npm run dev

## Seed Database

cd apps/api
npm run seed

## Run Tests

npm run test
